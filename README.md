# Installing

1. Clone this repo:
    
    ```
    $ git clone https://github.com/lucasfcosta/dotfiles --recursive
    ```

2. Give yourself permissions to run `./install.sh`
    
    ```
    $ chmod +x ./install.sh
    ```

3. Run `install.sh`
    
    ```
    $ sudo ./install.sh
    ```

4. Import gruvbox colors into iTerm
